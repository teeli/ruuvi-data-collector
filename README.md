# Ruuvi Data Collector

Scans for data from RuuviTag and RuuviAir devices and writes into [InfluxDB](https://www.influxdata.com/).

New and improved version of [ruuvi-influx-collector](https://github.com/teeli/ruuvi-influx-collector):

- Built with TypeScript on [Bun.sh](https://bun.sh/)
- Supports Ruuvi Air
